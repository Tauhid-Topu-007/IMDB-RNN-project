# IMDB Sentiment Analysis with RNN

A deep learning project that performs sentiment analysis on IMDB movie reviews using a Recurrent Neural Network (RNN) with SimpleRNN layers. The model classifies movie reviews as positive or negative with high accuracy.

## 📋 Project Overview

This project implements a sentiment analysis classifier using TensorFlow/Keras. It processes IMDB movie reviews and predicts whether the sentiment is positive or negative. The model uses an embedding layer followed by a SimpleRNN layer and a dense output layer with sigmoid activation.
